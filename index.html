Seattle Brewery Finder

First full-scale.

This was my first full-scale web app I built as a student in General Assembly's Web Development Immersive. We had a week to build it. This included wire-framing, working with a UX designer, and building/deploying the app. It is written in Node.js, with Express.js. as the framework. I used two API's: Mapbox for the map (which is built on Leaflet), and BreweryDB for brewery information. The app works with a Postgres database, and features user authentication.

This app is deployed using Heroku's free tier, and can take some time to load at first. Go to Seattle Brewery Finder here.
